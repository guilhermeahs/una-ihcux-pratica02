# una-ihcux-pratica02

Projeto prático em .NET criado para treinar a criação e execução de uma aplicação de console.

## Objetivo

O objetivo desta atividade foi praticar comandos básicos do .NET no terminal, como criação de projeto, execução e verificação da estrutura gerada automaticamente.

## Conteúdo do projeto

Este projeto contém:

- `Program.cs` → arquivo principal da aplicação
- `OlaMundo.csproj` → arquivo de configuração do projeto
- `README.md` → descrição do projeto
